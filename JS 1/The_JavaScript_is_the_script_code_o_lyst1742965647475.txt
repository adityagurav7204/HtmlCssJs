The JavaScript is the script(code or program) written in Java in HTML page.

The main motive of JavaScript is to process the web page.

The JavaScript code is written in head section of HTML page.

We must use <script> tag to wrap the JS code in HTML page.

The JavaScript has got the power to change the HTML content or even the CSS of a web page.
